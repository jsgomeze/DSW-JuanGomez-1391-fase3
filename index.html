Selectores CSS



Este tipo de selectores son usados para delimitar los elementos HTML que queremos aplicar estilos, para esto se puede ser muy preciso puesto que hay muchos estilos de etiquetas CSS.


El selector CSS es la primera regla CSS y funciona mediante una combinación o patrón de elementos con otros términos que le indican al investigador que elementos con otros términos que le indican al navegador que elementos HTML se les aplica la regla CSS, esta selección se llama sujeto del selector.


Este sirve para seleccionar etiquetas, es necesario conocer que cada tipo de CSS necesita ser admitido por el navegador, la mayoría de los selectores CSS, también están clasificados en especificación de nivel 3de selector, esta selección se llama sujeto del selector.


Estos tienen diferentes formas de aplicarse, pero si hay dos etiquetas con el mismo selector, para que sean mas legibles, lo mejor seria separar las dos etiquetas de renglón.



Tipos de selectores:




  • Selectores que delimita un elemento HTML, de tipo < h1>:

    • h1 {}


  • Selectores que delimitan una clase:

    • .box {}


  • Selectores que delimitan un ID:

    • #unique {}


  • Selección de atributo:

    • [title]{}


  • Selección de atributo que tiene un valor particular asignado:

    • a[href=”https://example.com”] {}
